## Overview  
MYTURTLE is an educational website that promotes symbolic sea turtle adoption.  
Its goal is to raise awareness about ocean protection and about the threats that sea turtles face (nets, hooks, pollution, habitat loss).

The project includes interactive components, informative sections, and a simple user experience designed to engage visitors.

---

## Website Structure  
The site is composed of several sections:

- **Hero section**: main mission  
- **Meet our turtles**: interactive carousel with six turtles  
- **Why adopt?**: explanation of adoption purpose  
- **WWF standards**: ethical framing of conservation  
- **About us**: mission and vision  
- **How it works**: steps of symbolic adoption  
- **Threats**: grid of major dangers for turtles  
- **Participation**: call to action  
- **Adopt Now**: adoption form  
- **Contact us**: email  
- **UniMi support**: map + institutional link  
- **Footer**

---

## Design Choices  
- Clean, modern layout  
- Consistent colors inspired by the sea, the turtles and the beaches
- Grid and flexbox for layout  
- Cards with images, shadows, rounded corners  
- Readable typography

---

## Technologies used  
**HTML** – page structure  
**CSS** – layout, styling, responsiveness  
**JavaScript** – interactive features:

- Cookie banner (accept/reject)
- Custom greeting for returning users
- User cookies stored with JavaScript
- Turtle carousel
- Adoption form with message confirmation

---

## Cookie Logic  
The site includes a custom cookie system:

- `cookie_consent` (save user’s choice for 30 days)  
- `username` (save visitor name for 1 day)  
- `last_self_localised_at` (save visitor location for 12 hours)  

A personalised greeting welcomes returning users.
